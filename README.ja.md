# pixel-texture-maker

https://code4fukui.github.io/pixel-texture-maker/

シンプレックスノイズとカラーグラデーションを使用して、プロシージャルなピクセルアートテクスチャを生成するブラウザベースのツールです。

![pixel-texture-maker アプリケーションインターフェースのスクリーンショット](https://user-images.githubusercontent.com/1715217/199686410-83bce132-7b56-46b4-b30f-0630d6a75897.jpg)

## 機能

- **カスタマイズ可能なパレット:** 備え付けのカラーピッカーを使用してカラーグラデーションを定義できます。
- **複数の色空間:** RGB、LAB、LCH、HSV、HSL、HSIの色空間で色を補間し、さまざまな視覚効果を得ることができます。
- **ノイズの制御:** コントラストとアニメーションのオフセットを調整する直感的なスライダーで、テクスチャの外観を調整できます。
- **アニメーション:** テクスチャが時間とともに変化する様子を観察できます。アニメーションはデフォルトで有効になっており、オン/オフの切り替えが可能です。
- **スタンドアロン:** 最新のブラウザ上で完全に動作し、インストールは不要です。

## 使い方

`index.html`を開くか、デモリンクにアクセスしてジェネレーターを使用します。インターフェースは256x256ピクセルのキャンバス（16x16のブロックで構成）と、いくつかのコントロールで構成されています。

- **カラーピッカー:** クリックして、テクスチャのグラデーションを形成する色を変更します。
- **モードドロップダウン:** グラデーションの補間に使用する色空間を選択します。
- **上部スライダー（コントラスト）:** ノイズ値に適用される指数を変更することで、色の分布を調整します。
- **下部スライダー（時間オフセット）:** ノイズパターンを手動でシークします。アニメーションが有効な場合、このスライダーは無効になります。
- **アニメーションボタン:** 自動アニメーションのオン/オフを切り替えます。

## 使用ライブラリ

このプロジェクトでは以下のオープンソースライブラリを使用しています。

- [chroma-es](https://github.com/code4fukui/chroma-es): 色変換やカラースケールのための軽量ライブラリ。
- [simplex-noise.js](https://github.com/jwagner/simplex-noise.js): 高速なシンプレックスノイズの実装。[SkyPack CDN](https://cdn.skypack.dev/)経由で読み込まれます。

## ライセンス

このアプリケーションで作成されたテクスチャデータは自由に使用できます。
